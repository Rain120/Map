# Map
电子地图管理系统
设计要求
    1.采用顺序表、链表、查找树（偷懒没写，哈哈哈）实现
    2.提供数据是二进制文件，读取二进制文件(.dat)，并保存在文本文档(.txt)中操作
    3.该程序功能包括
      1--读取文件（用户输入读取、保存路径）；
      2--插入（顺序插入、位置插入）;
      3--删除（根据LinkID删除）;
      4--查询（LinkID,Brunch,RoadName）;-----（实际要求还要分二分查找等查找方法，在设计的时候理解错误，最后偷懒没写）
      5--排序（顺序表7种，链表4种）;
      6--更新文件（将排序后的数据按照二进制数据存储回一个二进制文件(.dat)）
